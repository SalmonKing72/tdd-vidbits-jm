## TDD Capstone

### Vidbits

* setup with `npm install`
* from project root... ensure mongo db proces is running with `mongod --dbpath ./data/db`
* run tests with `npm test`
* run the app with `npm start`

### Tech.
* Mongodb
* ExpressJS
* Mongoose
* Handlebars
* Mocha, Chai, WebdriverIO JS, Supertest
